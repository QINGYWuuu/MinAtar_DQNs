This project is the simple implementation of value-based rl algorithms (including different dqns)

## Methods
  1. [DQN](http://www.nature.com/articles/nature14236)
  2. [Double DQN]
  3. [Maxmin DQN]
  4. [Averaged DQN]
  5. [Bootstrapped DQN]
  6. [Sunrise DQN]
  7. [Ensemble Voting DQN, UCB Exploration with Q-Ensembles]
  8.
